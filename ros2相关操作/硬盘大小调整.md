1.数据备份
```
`sudo apt install rsync -y
创建备份文件夹 
rsync -av /home/ ~/backup/home/
备份配置
rsync -av /etc/ ~/backup/etc/
```
2.移除无用包
```
sudo apt autoremove -y
sudo apt clean
# 清理用户缓存
rm -rf ~/.cache/*
# 清理日志
sudo journalctl --vacuum-time=2weeks

```
3.安装gui工具
```
sudo apt update sudo apt install parted gparted -y
sudo gparted
在界面：选择/dev/sda > 右键sda3 > “调整大小/移动” > 拖动滑块到最大（或输入新大小约90GB） > “调整大小/移动” > “应用”。
```
