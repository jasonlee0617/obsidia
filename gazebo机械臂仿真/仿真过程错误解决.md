#### rviz界面无法加载机械臂模型（Could not load library dlopen error: libsrdfdom.so.2.0.4: 无法打开共享目标文件: 没有那个文件或目录）
切到 ROS 库目录：
```
cd /opt/ros/humble/lib
```
给 2.0.7 版本创建一个 “2.0.4” 的链接：
```
sudo ln -s libsrdfdom.so.2.0.7 libsrdfdom.so.2.0.4
```
再刷新一次缓存：
```
sudo ldconfig
```

#### rviz界面ompl规划器找不到错误（找不到 `libompl.so.17`）
切到系统库目录
```
cd /lib/x86_64-linux-gnu
```
删除旧的、可能损坏的链接
```
sudo rm libompl.so.17
```
新建软连接
```
sudo ln -s libompl.so.16 libompl.so.17
ls -l libompl.so.17    # 应该看到 libompl.so.17 -> libompl.so.16
ldd /opt/ros/humble/lib/libmoveit_ompl_planner_plugin.so | grep ompl
```
