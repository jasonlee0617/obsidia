# Excalidraw Data

## Text Elements
1.图像获取和预处理： ^1cJeXn5K

ROS 订阅图像话题，获取 RGB 图像并通过 cv_bridge 转换为 OpenCV 格式。深度信息，处理深度图像获取与目标之间的距离。 ^KlUxPSJw

2.目标选择和初始化： ^bv4HZzho

通过鼠标在图像上选择目标区域（ROI），并初始化 KCF 跟踪器，KCF 跟踪器基于目标区域提取特征，获取目标区域的坐标信息，根据坐标信息获取深度信息，并开始跟踪目标。 ^h7fYvsSd

3.目标跟踪： ^LHJS94U5

KCF 跟踪器通过每帧图像更新目标的位置，并返回目标的矩形框坐标。通过 OpenCV 绘制矩形框在图像中显示跟踪结果。 ^5GZY0XV2

4.控制信号生成： ^L9FfpENi

通过目标位置的误差（距离误差和角度误差）计算线速度和角速度。根据这些速度值控制机器人向目标移动。 ^A21xrBTO

5.反馈与调整： ^QLhtJK6k

通过 PID 控制器，机器人根据目标位置的误差调整运动的线速度和角速度，确保机器人持续精确跟踪目标。 ^r1fgY1W2

6.用户交互与实时更新： ^L7dpNRg4

用户可以实时修改目标的选择区域（ROI），并更新跟踪器的目标区域。图像和目标状态会实时显示，以便用户查看和调整。 ^rmPSbouQ
