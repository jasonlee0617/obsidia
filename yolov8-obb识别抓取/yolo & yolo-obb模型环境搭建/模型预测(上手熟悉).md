**result[0]**输出分析：

```
ultralytics.engine.results.Results object
这是YOLOv8返回的标准结果对象，包含了检测的所有信息。
boxes:边界框信息
作用:存储所有检测到的目标的边界框信息
访问方式：`result[0].boxes.xyxy`（坐标）、`result[0].boxes.conf`（置信度）
```

```
keypoints: None
- **作用**：用于人体姿态估计任务
- **当前状态**：None（因为我们使用的是目标检测任务，不涉及关键点）
- **用途**：如果是姿态检测，会包含人体关节点坐标
```

```
masks: None
- **作用**：用于实例分割任务
- **当前状态**：None（目标检测任务不需要像素级分割）
- **用途**：如果是分割任务，会包含每个对象的像素掩码
```

```
names: {0: 'person', 1: 'bicycle', 2: 'car', ...}
- **作用**：将类别ID映射到类别名称
- **重要性**：COCO数据集的80个类别标准映射
- **使用**：当检测到类别ID为0时，对应'person'
- **常见类别解析**：
- 0: person（人）
- 1: bicycle（自行车）
- 2: car（汽车）
- 5: bus（公交车）
- 11: stop sign（停车标志）
- ...共80个类别
```

```
obb: None
- **作用**：有向边界框（Oriented Bounding Box）
- **当前状态**：None（标准检测使用水平边界框）
- **用途**：用于检测旋转物体
```

```
orig_img: array([[[119, 146, 172], [121, 148, 174], ...]])
- **格式**：NumPy数组
- **维度**：[高度, 宽度, 通道数(RGB)]
- **数值含义**：每个像素的RGB值（0-255）
- **示例解析**：[119, 146, 172] = RGB(119, 146, 172)
```

```
orig_shape: (1080, 810)
- **格式**：(高度, 宽度)
- **含义**：1080像素高，810像素宽
- **重要性**：用于坐标转换和结果还原
```

```
path: 'c:\\Users\\JasonLee\\OneDrive\\Desktop\\ultralytics-main\\ultralytics\\assets\\bus.jpg'
- **作用**：记录源图像的完整路径
- **用途**：便于溯源和批量处理
```

```
probs: None
- **作用**：用于分类任务的概率分布
- **当前状态**：None（检测任务不需要）
- **用途**：如果是分类任务，会显示各类别概率
```

```
save_dir: 'C:\\Users\\JasonLee\\OneDrive\\Desktop\\ultralytics-main\\runs\\detect\\predict9'
- **作用**：结果保存的目录路径
- **自动生成**：predict9表示第9次预测
- **内容**：包含标注后的图像和检测结果
```

```
- **preprocess**：预处理时间（2.9ms）
- **inference**：模型推理时间（18.9ms）
- **postprocess**：后处理时间（5.2ms）
- **总时间**：约27ms
```
