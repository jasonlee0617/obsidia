### 1.训练前准备
##### images:存放图片
train:训练集图片
val:验证集图片
test:测试集图片
##### labels:存放标签
train:训练集标签文件，要与训练集图片名称一一对应
val:验证集图片，要与验证集图片名称一一对应
test:测试集图片，要与验证集图片名称一一对应
### 2.模型训练
#### 1.创建yaml文件

**yolo-box-pen.yaml**

```
# Train/val/test sets as 1) dir: path/to/imgs, 2) file: path/to/imgs.txt, or 3) list: [path/to/imgs1, path/to/imgs2, ..]

path: box-pen  # dataset root dir
train: images/train  # train images (relative to 'path') 4 images
val: images/val  # val images (relative to 'path') 4 images
test:  # test images (optional)

# Classes:
names:
  0: box
  1: pen
  2:cube
  3:stone
```

#### 2.设置环境变量

```
conda activate yolov8 
conda env config vars set KMP_DUPLICATE_LIB_OK=TRUE 
conda deactivate 
conda activate yolov8
```

#### 3.命令行运行

设置环境变量：set KMP_DUPLICATE_LIB_OK=TRUE
或者
设置永久环境变量：
1.右键"此电脑" → "属性" → "高级系统设置" → "环境变量"
2.在"用户变量"中新建：
    - 变量名：`KMP_DUPLICATE_LIB_OK`
    - 变量值：`TRUE`

命令行代码：yolo task=detect mode=train model=./yolov8n.pt data=yolo-box-pen.yaml epochs=30 workers=1 batch=16

epochs=30:训练30轮
workers=0:必须设置为0

#### 4.代码形式运行

**创建yolo-box-pen-train.py**

```
from ultralytics import YOLO
# Load a model
model = YOLO('yolov8n.pt')

# Train the model

model.train(data='yolo-box-pen.yaml', workers=0, epochs=30, batch=16)
```

epochs=30:训练30轮
workers=0:必须设置为0
