### 2.安装pytorch
#### **一、PyTorch安装**

- **官方地址**：[https://pytorch.org/](https://pytorch.org/)
- **安装建议**：
    - 在一个独立的环境（如之前用`conda create -n yolov8`创建的环境）中，**能够使用`pip`就尽量使用`pip`**进行安装。
    - 只有在实在有问题的情况下（例如没有合适的、预编译好的系统版本安装包），再考虑使用`conda`安装。
    - **重要原则**：不要将`pip`和`conda`两种安装方式来回混合使用，以免导致环境混乱和依赖冲突。
#### **二、CUDA是否要安装**

- **如果只需要训练和简单推理**：则**无需**单独手动安装CUDA。直接通过PyTororch官网提供的命令安装预编译好的PyTorch（通常命令中已包含`cudatoolkit=xx.x`），即可自动获得运行所需的CUDA驱动库。
- **如果有部署需求**：例如需要将模型导出为**TensorRT**引擎模型，则**需要进行完整的CUDA Toolkit安装**。

#### **三、Pytorch安装注意事项（显卡与CUDA版本匹配）**

- **16XX系列显卡**：需要安装配套**CUDA 10.2（cu102）** 版本的PyTorch，否则可能导致训练出现问题。
- **30XX、40XX系列显卡**：需要安装**CUDA 11.1（cu111）及以上**版本的PyTorch，否则无法正常运行（因其需要更高版本的CUDA来支持新的架构特性）。

复制：pip install torch==1.13.1+cu116 torchvision==0.14.1+cu116 torchaudio==0.13.1 --extra-index-url https://download.pytorch.org/whl/cu116

### 6.训练无法使用GPU解决方法
#### Step 1：卸掉 CPU 版 torch
```
pip uninstall -y torch torchvision torchaudio
conda remove -y pytorch torchvision torchaudio
```

#### Step 2：安装 CUDA 版 PyTorch（推荐 cu124）
```
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu124
```

#### Step 3:安装完成后如何验证
```
python -c "import torch; print('torch:', torch.__version__); print('torch cuda:', torch.version.cuda); print('cuda available:', torch.cuda.is_available()); print('gpu:', torch.cuda.get_device_name(0) if torch.cuda.is_available() else 'None')"

```

✅ 正常结果应该类似：
- `torch: 2.x.x+cu124`
- `torch cuda: 12.4`
- `cuda available: True`
- `gpu: NVIDIA GeForce RTX 4060`
