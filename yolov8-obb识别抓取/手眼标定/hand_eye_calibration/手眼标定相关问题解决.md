

# 1.
”AttributeError: module 'cv2.aruco' has no attribute 'Dictionary_get'. Did you mean: 'Dictionary'?“
这是因为机器上装的 OpenCV 版本里，`cv2.aruco.Dictionary_get()` 已经被移除了／改名了，典型的解决方案是改用新的 API：
pip3 uninstall opencv-python opencv-contrib-python
pip3 install opencv-contrib-python==4.4.0.46

NumPy函数库版本需要在2.0以下
各个函数库的版本如下：
Python: 3.10.12 (main, May 27 2025, 17:12:29) [GCC 11.4.0]
NumPy:  1.23.0
pip install numpy==1.23.0
OpenCV: 4.6.0
pip install opencv-python==4.6.0.66
transforms3d: 0.4.2
需要安装 `opencv-contrib-python` 来启用 `cv2.aruco` 属性。
pip install opencv-contrib-python==4.6.0.66 --force-reinstall
